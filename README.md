# MECH309 term project: Data-based temperature prediction model
## Project Description
This project uses daily meteorological measurement in Montréal, QC to create a temperature prediction model. Different prediction horizon will be used. Models with different input parameters will be compared to each other, and a baseline model will be taken into account.

## Project Structure
- "/code" contains the Python C code
- "/plots" contains the output plots
- "/results" contains one summary table (for model 2 only)

Note: the requierments.txt indiquates the minimum versions of different libraries needed to run the project

## Instructions to run the code
After installing the necessary libraries, the code can be runed as follows:

- Clone the git repository on your computer
- Run the "/code/Project_Final_Version.py" file
- Depending on your needs, the dates can be changed to predict summer or winter values

Please be sure to run this file in the correct directory.
